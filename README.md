# Simple console md5 hash generator

## Supports

> Nodejs v6+

## Installation

* npm install
* yarn install

## Usage

* npm start
* yarn start
* node md5 [options] [command]

Options:
- -V, --version  output the version number
- -h, --help     output usage information

Commands:
- hash|h   make md5 hash
